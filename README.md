# PapaVoice 爸爸的声音胎教

A mobile-first pregnancy companion landing page for daily three-minute prenatal bonding with dad's voice.

## Features

- Warm hero section for Chinese, English, Italian, and French prenatal audio support.
- Today summary with unset pregnancy-week state and local-save messaging.
- Usage statistics for recording days, play count, and supported languages.
- Quick actions for playback and recording dad's voice.
- Local settings form for baby nickname, pregnancy week, dad's preferred title, reminder time, and today's message.
- Bottom tab navigation for Home, Recording, Playback, and Plan.

## Run locally

Open `index.html` directly in a browser, or serve the folder locally:

```bash
python3 -m http.server 4173
```

Then visit <http://127.0.0.1:4173/index.html>.
