PapaVoice

爸爸的声音胎教

先录入爸爸 1–3 分钟声音样本，部署到 Vercel 并设置 ElevenLabs API Key 后，可以把中文、English、Italiano、Français 胎教文案生成爸爸音色语音。

1. 录入爸爸声音 爸爸自然朗读 1–3 分钟，作为 AI 声纹样本。
2. 创建 AI 爸爸声纹 通过后端接口调用语音克隆服务，生成 voiceId。
3. 生成四国语音 中文 / English / Italiano / Français 都可以生成爸爸音色音频。
4. 手机声音库 生成后的声音保存到 IndexedDB，本机可反复播放。
上线要求

AI 声音转换必须部署到 Vercel，并在环境变量中设置 ELEVENLABS_API_KEY。GitHub Pages 不能运行后端接口，只能展示前端。
